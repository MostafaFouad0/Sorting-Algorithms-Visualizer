# Sorting Algorithms Visualizer
A program made with python and pygame that visualizes different sorting algorithms like:
- Heap Sort
- Selection Sort
- Bubble Sort
- Odd Even sort
- Insertion Sort
- Cocktail Sort

Will update it with more algorithms later
